Component Implementation Guide

Step-by-step instructions for building 3 key components from the Figma spec. Each guide includes the Figma preview, token mapping, and final code.

1 Button Component

1
Start with the base element 
<button class="btn btn-primary">Button Text</button>
2
Apply design tokens via CSS 
.btn {
  display: inline-flex;
  align-items: center;
  gap: var(--space-2);        /* 8px icon gap */
  height: 40px;
  padding: var(--space-2) var(--space-4);  /* 8px 16px */
  border: none;
  border-radius: var(--radius-md);  /* 8px */
  font-family: var(--font-family);
  font-size: var(--text-base);      /* 14px */
  font-weight: var(--font-semibold); /* 600 */
  line-height: var(--leading-base);  /* 24px */
  cursor: pointer;
  transition: all 0.15s ease;
}

.btn-primary {
  background: var(--brand);     /* #1F12DE */
  color: #fff;
}
.btn-primary:hover {
  background: var(--brand-hover); /* #1a0fbc */
}
.btn-primary:focus-visible {
  box-shadow: 0 0 0 4px var(--brand-light);
}
.btn-primary:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
3
Add variant classes for all states 
.btn-secondary {
  background: var(--bg-primary);
  color: var(--text-primary);
  border: 1px solid var(--border-primary);
}
.btn-secondary:hover {
  background: var(--bg-secondary);
}

.btn-destructive {
  background: var(--error-500);
  color: #fff;
}
.btn-destructive:hover {
  background: var(--error-700);
}

How to apply tokens
  1. Select any element on the canvas (text, frame, shape)
  2. In the Tokens Studio panel, find the token category (e.g., Colors)
  3. Right-click a token → choose where to apply it (Fill, Stroke, Border Radius, etc.)
  4. The element instantly updates with the token value
  5. If you later change the token value, all applied elements update automatically

Pro Tips
  • 1. Publish as a team library — Go to the Assets panel → click "Publish" → your team gets all tokens as shared styles across all files.
  • 2. Sync with Git — Tokens Studio supports GitHub, GitLab, and Bitbucket sync. Push token changes from Figma → auto-update your CSS/code.
  • 3. Dark mode switching — Enable both "openreel" and "dark" token sets → toggle between them to preview light/dark designs.
  • 4. Keep in sync — When tokens change in code, re-export tokens.json and re-import in Figma. All applied tokens update automatically.

Frequently Asked Questions

Can I edit tokens directly in Figma?
Yes! Once imported, you can modify any token value in the Tokens Studio panel. Changes apply to all elements using that token. You can then export the updated tokens back to JSON.
Do I need a paid Tokens Studio plan?
No — the free version handles local JSON import/export perfectly. Paid plans add Git sync, multi-file tokens, and team collaboration features.
What happens when I update the design system?
Download the latest tokens.json, re-import it in Tokens Studio, and click "Export to Figma" again. All styles update in place — no need to re-apply them.
Can I share this with my team?
Absolutely. After generating Figma styles, go to the Assets panel and click "Publish." Your entire team can then use these tokens across all their Figma files via the shared library.
