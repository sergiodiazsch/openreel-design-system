OpenReel DS 2.0 — Vision Overview

Three layers of upgrade: bug fixes, visual polish, and optional advanced effects

The Upgrade Plan

We're upgrading the design system across three tiers. Each builds on the previous one.

1

Tier 1 — Bug Fixes & Infrastructure

Fix what's broken. Token naming collisions in MCP, CSS extraction to shared file, semantic HTML in code generators, Svelte 5 syntax, Vue type mapping, deeper guidelines.

P0 Critical P1 High P2–P3 Medium
2

Tier 2 — Motion & Visual Polish

Make it feel alive. Smooth transitions, hover animations, input focus effects, colored accent shadows, micro-interactions on every component. Elegant and noticeable but not distracting.

Hover States Input Animations Colored Shadows Micro-interactions Page Transitions
3

Tier 3 — Advanced Effects (Optional Layer)

Premium showpieces. GSAP scroll-triggered animations, Three.js/WebGL particle backgrounds, morphing SVGs, parallax depth effects. Opt-in — not required, loaded only when needed.

GSAP ScrollTrigger Three.js Particles WebGL Shaders SVG Morphing Parallax

Architecture Approach

Everything layered and opt-in:

openreel-base.css — tokens + reset + component structure
openreel-motion.css — transitions, hovers, micro-interactions, colored shadows
openreel-effects.js — GSAP, Three.js, WebGL (lazy-loaded, opt-in)
