Design Principles

Clarity

Every element should have a clear purpose. If it doesn't communicate or help the user, remove it.

Consistency

Same problems get same solutions. Consistent patterns reduce cognitive load and build user trust.

Efficiency

Minimize steps to complete tasks. Respect users' time with smart defaults and streamlined flows.

Accessibility

Usable by everyone, always. WCAG 2.1 AA compliance is the baseline, not the aspiration.
