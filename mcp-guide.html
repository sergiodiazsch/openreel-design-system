Get Started in 60 Seconds

No coding required. Just copy, paste, done.

1

Install Claude Code

Download Claude Code from claude.ai/code and open it on your computer.

2

Copy this into your settings

Open Claude Code → Settings → MCP Servers → paste this:

{
  "mcpServers": {
    "openreel-ds": {
      "command": "npx",
      "args": [
        "-y",
        "github:sergiodiazsch/openreel-design-system/mcp"
      ]
    }
  }
}

Or clone the repo and point to the local path:

git clone https://github.com/sergiodiazsch/openreel-design-system.git
cd openreel-design-system/mcp && npm install
✓

That's it! Start building

Ask Claude: "Build me a dashboard using the OpenReel Design System" and it will use the right components, tokens, and patterns automatically.

No complicated setup. No config headaches.

1

Download the project

Grab the design system. Clone it with git, or just download the ZIP. 

git clone https://github.com/your-org/openreel-design-system.git
or Download ZIP
2

Run the setup script

One command does everything — installs dependencies, configures your AI tool, and verifies everything works. That's it. 

cd openreel-design-system && ./setup.sh

What this does: Installs Node.js packages, sets up the MCP server, and creates the config file your AI tool needs. You don't need to do anything else.

Prefer to set up manually? See IDE-specific instructions below.
3

"Create a login page with social auth buttons"

"Design a dashboard with stat cards and a data table"

"Make a pricing page with three plan cards"

Manual IDE setup

If you prefer to configure things yourself instead of using setup.sh, expand your IDE below.

First install dependencies, then add to .claude/settings.json in your project: 

cd mcp && npm install
{
  "mcpServers": {
    "openreel-ds": {
      "command": "node",
      "args": ["/path/to/openreel-design-system/mcp/index.js"]
    }
  }
}

Replace /path/to with the actual path on your machine.

Install dependencies, then add to .cursor/mcp.json in your project: 

cd mcp && npm install
{
  "mcpServers": {
    "openreel-ds": {
      "command": "node",
      "args": ["/path/to/openreel-design-system/mcp/index.js"]
    }
  }
}

Replace /path/to with the actual path on your machine.

Install dependencies, then add to your Windsurf MCP settings: 

cd mcp && npm install
{
  "mcpServers": {
    "openreel-ds": {
      "command": "node",
      "args": ["/path/to/openreel-design-system/mcp/index.js"]
    }
  }
}

Replace /path/to with the actual path on your machine.

For any MCP-compatible tool, use the standard stdio transport configuration: 

cd mcp && npm install
{
  "mcpServers": {
    "openreel-ds": {
      "transport": "stdio",
      "command": "node",
      "args": ["/path/to/openreel-design-system/mcp/index.js"]
    }
  }
}

The server communicates over stdin/stdout using JSON-RPC. Any MCP client will work.
