WCAG 2.5.5 Reference

The minimum target size for pointer inputs must be at least 24x24 CSS pixels (Level AA). For enhanced accessibility (Level AAA), aim for at least 44x44 CSS pixels. This ensures users with motor impairments or those using touch screens can reliably interact with controls.
