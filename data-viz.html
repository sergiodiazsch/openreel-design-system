Accessibility Guidelines for Data Viz

          
Ensure all users can perceive and understand data visualizations.

        

        

          

            

              
Best Practices

              
    
                
  • Don't rely on color alone -- use patterns, textures, or labels
    • 
                
  • Provide data tables as accessible alternatives to charts
    • 
                
  • Announce data changes with ARIA live regions for screen readers
    • 
                
  • Maintain minimum 3:1 contrast for chart elements against background
    • 
                
  • Ensure 4.5:1 contrast for text labels in and around charts
    • 
                
  • Use role="img" and aria-label on SVG chart containers
    • 
                
  • Support keyboard navigation for interactive chart elements
    •
